开始学习 C++ 之前的一些认知。

C++ 语言特点

C++ 融合了 3 种不同的编程方式：

  • C 语言代表的过程性编程。
  • C++ 在 C 语言基础上添加的类代表的面向对象（OOP）编程。
  • C++ 模板支持的泛型编程。

C++ 起源

诞生于贝尔实验室，Bjarne Stroustrup 于 20 世纪 80 年代开发出了这种语言。基于 C 语言。

C++ 标准

  • 移植时需要对程序重新编译。
  • 常用的两种标准，C++98, C++11 标准。
